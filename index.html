How to Deploy This Library

1
Create a GitHub repo

Go to github.com, create a new public repository called mba-primer-library. Initialize with a README.

2
Upload your files

Upload index.html and all four .docx files into the repo root. File names must match exactly as listed in each card.

3
Enable GitHub Pages

Go to Settings → Pages → Source: Deploy from branch → Branch: main → Folder: / (root) → Save.

4
Share the link

Your library goes live at username.github.io/mba-primer-library — share with classmates. Free, permanent, no login required.
